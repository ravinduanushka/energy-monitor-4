# Smart Energy Usage Monitor ⚡

A professional-grade energy monitoring simulation built with **C++** and **Modern Web Technologies**.

## 🚀 Quick Start
1. Run `build.bat` to compile the engine.
2. Run `run.bat` to choose between the **C++ Console** or the **Web Dashboard**.

## 🏗️ Architecture & Data Structures
This project is built using 7 specialized data structures for maximum efficiency:
*   **Graph**: Room topology and DFS analysis.
*   **BST**: Rapid power-range searching.
*   **Hash Map**: Instant device lookup.
*   **Stack**: Action history (Undo feature).
*   **Queue**: Real-time data processing.
*   **Linked List**: Alert history management.
*   **Sliding Window**: Energy trend prediction.
